Apple Music User Token Generator

Music User Token (MUT) を取得するためのツールです。初回セットアップ時に1回だけ使用します。

ローカルで生成した JWT Developer Token を入力してください。
生成方法: python -c "import jwt, time; print(jwt.encode({'iss':'YOUR_TEAM_ID','iat':int(time.time()),'exp':int(time.time())+3600}, open('path/to/AuthKey.p8').read(), algorithm='ES256', headers={'kid':'YOUR_KEY_ID'}))"

Apple Music のサブスクリプションが有効な Apple ID でサインインしてください。

このトークンを GitHub Settings → Secrets → APPLE_MUSIC_USER_TOKEN に設定してください。
有効期限は約6ヶ月です。期限切れ時はこのページで再取得してください。
